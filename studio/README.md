# Sanity Studio - Isyraq Annur

Untuk saat ini, kita akan menggunakan **Sanity Studio yang di-host oleh Sanity.io** langsung, bukan lokal.

## Cara Akses Studio:

1. Buka browser dan kunjungi: **https://www.sanity.io/manage**
2. Login dengan akun Google Anda (`fuyudunniam@gmail.com`)
3. Pilih project **"IsyraqAnnur"** (Project ID: `yamgwplz`)
4. Klik **"Vision"** atau **"Content"** untuk mulai mengelola data

## Keuntungan Menggunakan Hosted Studio:

✅ Tidak perlu setup lokal yang rumit  
✅ Bisa diakses dari mana saja  
✅ Otomatis ter-update ke versi terbaru  
✅ Tidak perlu maintain server lokal  

## Schema yang Sudah Disiapkan:

Berikut schema yang sudah saya buat dan siap digunakan:

1. **Blog Post** - Artikel dan berita kegiatan
2. **Author** - Profil penulis
3. **Category** - Kategori artikel
4. **Testimonial** - Testimoni donatur/wali santri
5. **Partner** - Logo dan info mitra kerjasama
6. **Achievement** - Galeri prestasi santri

## Langkah Selanjutnya:

Untuk mengaktifkan schema di project Sanity Anda, kita perlu deploy schema ini ke Sanity Cloud. Saya akan bantu setup-nya nanti setelah Anda berhasil akses dashboard Sanity.

---

**Catatan**: Jika nanti Anda ingin Studio lokal yang berfungsi sempurna, kita bisa setup ulang dengan template resmi Sanity yang lebih stabil.
